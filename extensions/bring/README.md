# Bring!

Manage your Bring! shopping list via Raycast.

## Obtaining Authentication Tokens
1. Log in to your web [Bring!](https://web.getbring.com/) account.
2. Open the Web Inspector, and go to the "Application" tab.
3. Click open the `Local Storage` dropdown on the sidebar, under storage, and click on the option that says `https://web.getbring.com/`.
4. Look for and copy the value labeled `accessToken`.
5. Moreover, look for and copy the value labeled `userUuid`.

That's it! Now, put `accessToken` and `userUuid` in the setup screen and you're all good!
